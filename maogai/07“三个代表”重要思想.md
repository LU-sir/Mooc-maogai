# 07“三个代表”重要思想

1 【单选题】冷战结束后,世界的主题是(    )   

    * A、战争与革命
    * B、和平与发展
    * C、经济全球化
    * D、世界多极化
正确答案： B    

2 【单选题】江泽民首次对“三个代表”进行比较全面的阐释的时间是(   )   

    * A、1989年6月
    * B、1997年9月
    * C、2000年2月
    * D、2001年7月
